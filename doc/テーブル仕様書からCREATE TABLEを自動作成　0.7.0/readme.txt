このツールは、Excelに記述されているテーブル仕様書から、CREATE TABLE文のSQLを自動生成するためのものです。

現時点では、以下の部分を対応していません。
・DEFAULT制約
・CHECK制約
・テンポラリテーブル
・制約違反時のアルゴリズム

これらに関しては、ご要望があれば対応いたします。
連絡先は、以下のとおりです。

開発者：いかちソフトウェア
URL：http://www.ikachi.org/
MAIL:ikachisoftware@gmail.com

使い方についての説明は、Excelファイルに掲載しています。
